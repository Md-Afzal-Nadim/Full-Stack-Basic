# 📝 Notes Management App – MERN Stack

A full-stack Notes Management application built using **React JS, Node.js, Express.js, and MongoDB**.  
This project allows users to create, update, delete, and manage notes with real-time database changes.

The frontend and backend are deployed on **Render**, and the application is fully responsive for **mobile and desktop devices**.

## ✨ Features

✅ Create New Notes  
✅ Update Existing Notes  
✅ Delete Notes  
✅ Store Data in MongoDB  
✅ Real-time CRUD Operations  
✅ Responsive Design (Mobile + Laptop)  
✅ Clean User Interface  
✅ Fast API Integration  
✅ Full Stack MERN Architecture  
✅ Frontend + Backend Deployment on Render

---

## 📚 Functionalities

### Add Note
- Enter Title  
- Enter Description  
- Save note to MongoDB

### Update Note
- Edit title or description  
- Instantly update database

### Delete Note
- Remove notes permanently
